# Rede Neurais Artificiais + Python
Utilização da biblioteca PyBrain para a implementação de Redes Neurais Artificiais

## Redes implementadas

- Perceptron (XOR)
- MLP com Backpropagation (Treinando uma base de dados a partir de um arquivo txt)
